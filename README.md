# UNP_PROJECT_E_COMMERCE_DATA_ANALYSIS_USING_MySQL
I did this project in collaboration with UNP. The project focuses on designing and implementing a comprehensive e-commerce database system using SQL to manage and organize various operations involved in an online shopping platform. A database named smart_cart was developed to handle key components such as customer information, product details, order transactions, payments, returns, shipments, employee management, and cart activities.
The system was structured by creating multiple relational tables, each defined with appropriate primary keys to ensure uniqueness and data integrity. 
Logical relationships were established between tables to reflect real-world interactions, such as linking customers with their orders and products, and tracking the complete lifecycle of an order from purchase to delivery and return.
Data was systematically populated across all tables to represent realistic transaction flows, enabling effective retrieval and analysis. Various SQL operations were performed, including SELECT queries for data retrieval, JOIN operations to combine related data from multiple tables, WHERE clauses with logical operators for precise filtering, GROUP BY for aggregating data, and ORDER BY for sorting results in a meaningful manner.
Through these queries, insights such as customer purchasing patterns, order distribution, product pricing trends, and transaction details were examined. The project demonstrates how structured databases can efficiently manage interconnected data and support analytical operations.
Overall, this project provided practical exposure to database design, query execution, and data analysis within an e-commerce context, highlighting the importance of SQL in handling real-time business data and supporting informed decision-making.
